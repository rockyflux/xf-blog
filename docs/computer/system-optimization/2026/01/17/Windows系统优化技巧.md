# Windows 系统优化技巧

<!-- more -->

## 关闭不必要的启动项

1. 打开任务管理器
2. 切换到"启动"标签
3. 禁用不需要的程序

## 清理系统垃圾

使用磁盘清理工具：
1. 右键点击 C 盘
2. 选择"属性"
3. 点击"磁盘清理"

## 优化虚拟内存

1. 右键"此电脑" -> 属性
2. 高级系统设置
3. 性能设置 -> 高级
4. 虚拟内存设置

## 关闭视觉效果

对于性能较低的电脑，可以关闭一些视觉效果来提高性能。

## 总结

定期优化系统可以保持电脑的良好性能。
