# Windows 必备软件推荐

<!-- more -->

## 系统工具

### Everything
- 文件搜索工具
- 秒搜全盘文件
- 轻量级，占用资源少

### Dism++
- 系统优化工具
- 清理系统垃圾
- 系统备份还原

## 开发工具

### VS Code
- 轻量级代码编辑器
- 丰富的插件生态
- 跨平台支持

### Git
- 版本控制工具
- 必备开发工具

## 效率工具

### Snipaste
- 截图工具
- 贴图功能
- 标注功能强大

### QuickLook
- 空格键预览文件
- 支持多种格式
- 提高工作效率

## 总结

这些软件可以大大提高 Windows 使用体验和工作效率。
