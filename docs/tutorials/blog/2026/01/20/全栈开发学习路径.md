# 全栈开发学习路径

<!-- more -->

## 前端技能

### 基础技能

- HTML/CSS/JavaScript
- 响应式设计
- 浏览器兼容性

### 进阶技能

- 前端框架（Vue/React/Angular）
- 状态管理
- 构建工具（Webpack/Vite）

## 后端技能

### 基础技能

- 编程语言（Java/Node.js/Python）
- 数据库（MySQL/MongoDB）
- API 设计

### 进阶技能

- 微服务架构
- 缓存技术
- 消息队列

## 学习建议

1. 先掌握前端或后端其中一个方向
2. 逐步学习另一个方向
3. 通过项目实践巩固知识
4. 关注新技术和最佳实践

## 总结

全栈开发需要掌握前后端知识，建议循序渐进地学习。
