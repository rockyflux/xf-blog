# pnpm 使用指南

<!-- more -->

## 安装

```bash
npm install -g pnpm
```

## 基本命令

### 安装依赖
```bash
pnpm install
pnpm add <package>
```

### 运行脚本
```bash
pnpm run <script>
```

## 优势

- 更快的安装速度
- 节省磁盘空间
- 严格的依赖管理

## 总结

pnpm 是一个高效的包管理工具，特别适合大型项目。
