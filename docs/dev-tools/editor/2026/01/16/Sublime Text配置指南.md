# Sublime Text 配置指南

<!-- more -->

## 安装 Package Control

Sublime Text 的包管理器，必须先安装。

1. 打开命令面板（Ctrl+Shift+P）
2. 输入 `Install Package Control`
3. 等待安装完成

## 推荐插件

### Emmet
快速编写 HTML/CSS 代码。

### SideBarEnhancements
增强侧边栏功能。

### BracketHighlighter
高亮匹配的括号。

## 配置技巧

### 自定义快捷键
在 `Preferences -> Key Bindings` 中配置。

### 主题设置
推荐使用 `Material Theme`。

## 总结

合理配置 Sublime Text 可以大大提高编码效率。
