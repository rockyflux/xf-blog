# VS Code 常用插件推荐

<!-- more -->

## 必备插件

### 1. Chinese (Simplified) Language Pack
VS Code 的中文语言包，让界面更友好。

### 2. GitLens
强大的 Git 增强工具，可以查看代码历史、作者信息等。

### 3. Prettier
代码格式化工具，支持多种语言，保持代码风格统一。

## 开发效率插件

### 4. Auto Rename Tag
自动重命名配对的 HTML/XML 标签。

### 5. Bracket Pair Colorizer
为匹配的括号添加颜色，提高代码可读性。

### 6. Path Intellisense
自动补全文件路径。

## 总结

这些插件可以大大提高开发效率，建议根据实际需求选择合适的插件。
