# Vim 基础操作

<!-- more -->

## 模式切换

- `i` - 进入插入模式
- `Esc` - 退出到普通模式
- `:` - 进入命令模式

## 基本移动

- `h` `j` `k` `l` - 左、下、上、右
- `w` - 移动到下一个单词
- `b` - 移动到上一个单词

## 编辑操作

- `dd` - 删除当前行
- `yy` - 复制当前行
- `p` - 粘贴
- `u` - 撤销

## 保存退出

- `:w` - 保存
- `:q` - 退出
- `:wq` - 保存并退出
- `:q!` - 强制退出不保存

## 总结

掌握这些基础操作，可以大大提高在终端中的编辑效率。
