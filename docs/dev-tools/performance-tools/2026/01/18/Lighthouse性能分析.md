# Lighthouse 性能分析

<!-- more -->

## 使用方法

1. 打开 Chrome DevTools
2. 切换到 Lighthouse 标签
3. 选择分析类型
4. 点击"生成报告"

## 评分指标

- Performance - 性能
- Accessibility - 可访问性
- Best Practices - 最佳实践
- SEO - 搜索引擎优化

## 优化建议

Lighthouse 会提供具体的优化建议，按照建议逐步改进。

## 总结

Lighthouse 是网站性能分析的重要工具，定期检查可以保证网站质量。
