# Chrome Performance 分析

<!-- more -->

## 录制性能

1. 打开 DevTools
2. 切换到 Performance 标签
3. 点击录制按钮
4. 执行操作
5. 停止录制

## 分析结果

- FPS - 帧率
- CPU - CPU 使用率
- 网络请求时间线
- 内存使用情况

## 优化建议

根据性能分析结果，找出瓶颈并进行优化。

## 总结

Chrome Performance 工具可以深入分析页面性能，找出性能问题。
