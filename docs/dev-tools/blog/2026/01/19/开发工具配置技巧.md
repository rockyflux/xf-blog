# 开发工具配置技巧

<!-- more -->

## 编辑器配置

### VS Code 配置优化

VS Code 的配置可以通过 `settings.json` 文件进行自定义，以下是一些常用配置：

```json
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  "files.autoSave": "afterDelay"
}
```

### 快捷键配置

合理配置快捷键可以大大提高编码效率，建议根据个人习惯进行自定义。

## 终端配置

### PowerShell 主题配置

可以使用 Oh My Posh 来美化 PowerShell 终端，提供更好的视觉体验。

## 总结

合理的工具配置可以让开发工作更加顺畅，建议定期优化和调整配置。
