# 开发工具插件推荐

<!-- more -->

## VS Code 插件

### GitLens

GitLens 是一个强大的 Git 可视化插件，可以显示代码的 Git 历史信息。

### Prettier

Prettier 是一个代码格式化工具，支持多种编程语言，可以自动格式化代码。

### ESLint

ESLint 是一个 JavaScript 代码检查工具，可以帮助发现代码中的潜在问题。

## 浏览器插件

### Vue DevTools

Vue DevTools 是 Vue.js 开发的必备工具，可以方便地调试 Vue 应用。

## 总结

合适的插件可以扩展开发工具的功能，提高开发效率。
