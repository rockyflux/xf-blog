# Navicat 使用技巧

<!-- more -->

## 连接数据库

1. 点击"新建连接"
2. 选择数据库类型
3. 填写连接信息
4. 测试连接

## 常用功能

### 查询编辑器
- 语法高亮
- 自动补全
- 执行计划

### 数据同步
- 结构同步
- 数据同步
- 同步设置

## 快捷键

- `Ctrl+Enter` - 执行查询
- `Ctrl+Shift+C` - 复制为 SQL
- `F5` - 刷新

## 总结

Navicat 是功能强大的数据库管理工具，掌握其使用技巧很重要。
