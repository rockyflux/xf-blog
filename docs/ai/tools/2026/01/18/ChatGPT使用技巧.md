# ChatGPT 使用技巧

<!-- more -->

## 提示词工程

### 明确你的需求
好的提示词应该：
- 清晰明确
- 包含上下文
- 指定输出格式

### 示例
```
请帮我写一个 Python 函数，功能是：
1. 接收一个列表作为参数
2. 返回列表中所有偶数的和
3. 包含错误处理
4. 添加函数注释
```

## 角色扮演

让 ChatGPT 扮演特定角色：
```
你是一位经验丰富的 Python 开发工程师，请帮我...
```

## 分步骤思考

使用 Chain of Thought 技巧：
```
请分步骤思考并解决以下问题：
1. 首先分析问题
2. 然后列出解决方案
3. 最后给出具体实现
```

## 迭代优化

如果第一次回答不满意：
- 指出具体问题
- 要求更详细的解释
- 要求提供示例

## 总结

掌握这些技巧，可以让 ChatGPT 更好地为你服务。
