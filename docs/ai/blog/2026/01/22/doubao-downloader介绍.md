# Doubao Downloader：抖音视频下载工具

## 项目简介

Doubao Downloader 是一个专为抖音平台设计的视频和图片下载工具，支持从抖音网页端和手机端分享链接中批量下载无水印视频和图片。该工具提供了便捷的下载体验，特别适合内容创作者、研究人员和普通用户使用。

## 主要功能特性

### 🎥 视频下载
- **无水印下载**：自动去除抖音视频的水印，提供原始质量的视频
- **批量下载**：支持同时下载多个视频，提高效率
- **顺序下载**：可选择按顺序下载，确保下载顺序与选择顺序一致
- **手机端支持**：支持捕获手机端分享的视频链接

### 🖼️ 图片下载
- **图片检测**：自动检测捕获的图片是否包含水印
- **压缩打包**：将多张图片打包成ZIP文件，便于管理和传输
- **进度显示**：实时显示下载进度，让用户了解下载状态

### 🔧 技术特性
- **版本自动检查**：基于GitHub releases latest API，自动检查更新
- **本地存储优化**：使用IndexDB代替localStorage，提供更好的存储性能
- **并发下载控制**：使用p-limit库实现并发下载，避免浏览器限制
- **虚拟滚动**：针对大量图片渲染进行优化，提升界面流畅度
