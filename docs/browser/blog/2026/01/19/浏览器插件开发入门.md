# 浏览器插件开发入门

<!-- more -->

## 插件开发基础

### Manifest文件

Manifest 文件是插件的配置文件，定义了插件的基本信息。

### 权限声明

在 Manifest 中声明插件需要的权限。

## 开发流程

### 创建项目结构

```
my-extension/
  manifest.json
  popup.html
  content.js
  background.js
```

### 编写代码

- popup.html: 插件弹窗界面
- content.js: 内容脚本
- background.js: 后台脚本

## 测试和发布

### 本地测试

在浏览器中加载未打包的扩展进行测试。

### 发布到商店

完成开发后可以发布到 Chrome Web Store 或 Firefox Add-ons。

## 总结

浏览器插件开发相对简单，是学习 Web 开发的好方式。
