# Chrome 快捷键大全

<!-- more -->

## 标签页操作

- `Ctrl + T` - 新建标签页
- `Ctrl + W` - 关闭当前标签页
- `Ctrl + Tab` - 切换到下一个标签页
- `Ctrl + Shift + Tab` - 切换到上一个标签页

## 页面操作

- `Ctrl + R` - 刷新页面
- `Ctrl + Shift + R` - 强制刷新
- `Ctrl + F` - 页面内搜索
- `F11` - 全屏模式

## 地址栏

- `Ctrl + L` - 聚焦地址栏
- `Alt + D` - 聚焦地址栏

## 开发者工具

- `F12` - 打开开发者工具
- `Ctrl + Shift + I` - 打开开发者工具
- `Ctrl + Shift + J` - 打开控制台

## 总结

掌握这些快捷键可以大大提高浏览效率。
