# Chrome 必备插件推荐

<!-- more -->

## 广告拦截

### uBlock Origin
- 强大的广告拦截器
- 轻量级，不占用资源
- 自定义规则

## 翻译工具

### 沙拉查词
- 划词翻译
- 支持多种词典
- 学习模式

## 开发工具

### Vue.js devtools
- Vue 开发调试工具
- 组件树查看
- 状态管理调试

### React Developer Tools
- React 开发调试工具
- 组件检查
- 性能分析

## 效率工具

### Tampermonkey
- 用户脚本管理器
- 自定义网页功能
- 强大的扩展能力

### LastPass
- 密码管理器
- 自动填充
- 安全可靠

## 总结

这些插件可以大大提高浏览器的使用体验和工作效率。
