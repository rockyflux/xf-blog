# 内容管理助手

## 内容组织结构

### 目录结构规范
```
docs/
├── ai/                    # AI相关内容
├── dev-tools/            # 开发工具
├── tutorials/            # 教程文档
├── basics/               # 基础知识
├── computer/             # 计算机技术
├── browser/              # 浏览器技术
├── online-tools/         # 在线工具
├── tools/                # 通用工具
├── index.md             # 首页
├── tags.md              # 标签页
└── archives.md          # 归档页
```

### 文章路径规则
- 格式：`分类/YYYY/MM/DD/文章标题.md`
- 示例：`docs/ai/tools/2026/01/20/AI编程工具汇总对比.md`
- 日期格式：YYYY/MM/DD
- 文件名：使用描述性英文或中文

## 发布流程

### 新文章发布
1. **确定分类**：选择合适的分类目录
2. **创建文件**：按日期格式创建 Markdown 文件
3. **编写内容**：按照文档规范编写
4. **添加元数据**：完整的 frontmatter
5. **更新索引**：添加到侧边栏配置
6. **预览测试**：本地预览确认效果
7. **提交发布**：推送到主分支触发自动部署

### 内容更新
1. **版本控制**：记录修改历史
2. **内容审核**：检查准确性和完整性
3. **链接更新**：确保内部链接有效
4. **索引同步**：更新相关索引页面

## 元数据管理

### Frontmatter 标准字段
```yaml
---
title: 文章标题
description: 简短描述，SEO友好
date: 2026-01-20
tags: [标签1, 标签2, 标签3]
author: 箫风
category: 分类名称
---
```
